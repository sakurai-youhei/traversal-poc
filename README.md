# traversal-poc

## Version 2

**MANAGER**

```
curl -sSL https://github.com/sakurai-youhei/traversal-poc/raw/main/v2/manager.py | sudo python3 <agent> [<agent> ...]
```

**AGENT**

```
curl -sSL https://github.com/sakurai-youhei/traversal-poc/raw/main/v2/agent.py | sudo python3 <manager>
```

**Prerequisites**

- sudo
- iptables
- curl
- Python 3.7 or later
